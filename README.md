# NutriShare AI - Smart Food Sustainability System

NutriShare AI is an automated solution designed to bridge the gap between food surplus creators (restaurants, mess halls, marriage lawns) and food distributors (NGOs, local shelters). This project directly aligns with **UN SDG 2 (Zero Hunger)** and **UN SDG 12 (Responsible Consumption and Production)**.

## Key Features
- **Predictive Analytics:** Uses simple machine learning models (Regression via IBM BOB) to forecast daily food waste based on historical attendance trends.
- **Automated Routing:** Optimizes the collection path for volunteers to pick up fresh food before it spoils.
- **Instant SMS/WhatsApp Alerts:** Notifies the nearest shelter automatically when a donor confirms surplus food.

## AI Implementation & Tools
- **Data Modeling:** Built and tested using IBM BOB Data Tools for predictive trends.
- **Routing Algorithm:** Implemented using Python's optimization libraries (scikit-learn and pulp).
